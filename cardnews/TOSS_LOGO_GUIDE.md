# 토스 로고 사용 가이드

## ✅ 사용 가능한 경우

### 1. 토스 앱인토스 등록 앱
**상태:** ✅ **가능**

**이유:**
- 배달수수료 계산기는 **토스 앱인토스에 승인된 미니앱**입니다.
- 토스 플랫폼 내에서 운영되는 공식 앱이므로, 토스 생태계의 일부로 인정됩니다.
- 마케팅 자료에서 "토스에서 사용 가능"을 표시하는 것은 **일반적으로 허용**됩니다.

### 2. 사용 가능 범위
**✅ 허용되는 경우:**
- "토스에서 이용 가능"
- "토스 미니앱"
- 토스 로고 + "앱 제공 플랫폼" 표시
- 인스타/SNS 카드뉴스에서 "토스 미니앱" 표기

**❌ 금지되는 경우:**
- 토스가 직접 개발/운영한다는 오해 유발
- "토스 공식" 표기 (공식 파트너십 없이)
- 토스 로고를 변형하거나 왜곡
- 토스가 추천/보증한다는 암시

## 📋 권장 사용 방법

### 카드뉴스 마지막 컷 (CTA)

**옵션 1: 토스 로고 + 앱명 (권장)**
```
[토스 로고]
-------------------
배달수수료 계산기

토스 미니앱에서 이용 가능
```

**옵션 2: 텍스트만 (안전)**
```
배달수수료 계산기

토스 앱인토스에서 사용하세요
프로필 링크 클릭
```

**옵션 3: 토스 로고 작게 (가장 안전)**
```
배달수수료 계산기
프로필 링크 클릭

[하단 작은 글씨]
Available on [토스 로고]
```

## ⚖️ 법적 리스크 체크

### 낮은 리스크 ✅
- 토스 앱인토스에 **정식 승인된 앱**
- "토스에서 이용 가능" 표기
- 토스 로고를 원본 그대로 사용
- 토스가 플랫폼 제공자임을 명시

### 중간 리스크 ⚠️
- 토스 로고를 크게 강조
- "토스 파트너" 같은 모호한 표현
- 공식 파트너십 없이 "추천" 암시

### 높은 리스크 ❌
- 토스가 직접 개발했다고 오해 유발
- "토스 공식" 표기
- 토스 로고 변형/왜곡
- 토스 브랜드 이미지 훼손

### 1단계: 안전하게 시작
**카드뉴스 마지막 컷:**
```
배달수수료 계산기
토스 미니앱에서 이용 가능

[하단 작게 토스 로고]
```

**텍스트:**
- "토스 앱인토스에서 사용하세요"
- "토스에서 바로 계산"
- "토스 미니앱 '배달수수료 계산기'"

### 2단계: 필요 시 확인
토스 앱인토스 콘솔에서 "마케팅 자료 가이드" 확인하거나,
토스 파트너 지원팀에 문의:
- 이메일: partner@toss.im (추정)
- 앱인토스 콘솔 내 문의 기능

## 📌 결론

**✅ 토스 로고 사용 가능합니다!**

**조건:**
1. 토스가 **플랫폼 제공자**임을 명시
2. 로고 원본 그대로 사용 (변형 금지)
3. "토스 공식" 같은 오해 유발 표현 자제
4. 작게 표시 (앱 브랜드가 메인, 토스는 플랫폼 표시)

**카드뉴스 권장 구성:**
- 상단 작게: [토스 로고]
- 메인: 배달수수료 계산기
- 하단: "토스 미니앱에서 이용 가능"

이렇게 하면 **법적 문제 없이** 토스의 신뢰도를 활용할 수 있습니다! 🎯
