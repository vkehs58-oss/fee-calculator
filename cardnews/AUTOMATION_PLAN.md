## 🔄 자동화 파이프라인

### 1단계: 뉴스 수집 (자동)
**스크립트:** `auto_cardnews_system.py`
**실행:** 매일 오전 9시
**기능:**
- 네이버 뉴스 크롤링 (배민/쿠팡 키워드)
- 사장님 관점 필터링
- 카드뉴스 후보 3개 선정
- 대본 초안 자동 생성
- 텔레그램 알림

### 2단계: 대본 검토 (수동, 30분)
**담당:** 사장님
**작업:**
- 옥타가 보낸 대본 3개 중 1개 선택
- 카피 수정 (필요 시)
- 제미나이에게 배경 이미지 요청

### 3단계: 이미지 제작 (제미나이, 1시간)
**담당:** 제미나이
**작업:**
- 컷별 배경 이미지 생성 (8컷)
- 어두운 배경 + 빨강/노랑 톤
- 마지막 컷: 파란색 + 토스 로고 공간

### 4단계: 합성 (수동, 30분)
**담당:** 사장님
**작업:**
- CapCut에서 텍스트 추가
- 토스 로고 배치
- 1080x1080 저장

### 5단계: 발행 (수동, 10분)
**담당:** 사장님
**작업:**
- 인스타 피드 발행
- 캡션 붙여넣기
- 해시태그 추가

## 🤖 LaunchAgent 설정

### 매일 오전 9시 자동 실행

**파일 위치:** `~/Library/LaunchAgents/com.boss.cardnews.plist`

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.boss.cardnews</string>
    
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/python3</string>
        <string>/Users/junb/Desktop/boss/fee-calculator/cardnews/auto_cardnews_system.py</string>
    </array>
    
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>9</integer>
        <key>Minute</key>
        <integer>0</integer>
    </dict>
    
    <key>StandardOutPath</key>
    <string>/Users/junb/Desktop/boss/fee-calculator/cardnews/logs/cardnews.log</string>
    
    <key>StandardErrorPath</key>
    <string>/Users/junb/Desktop/boss/fee-calculator/cardnews/logs/cardnews_error.log</string>
</dict>
</plist>
```

**설치 명령:**
```bash
launchctl load ~/Library/LaunchAgents/com.boss.cardnews.plist
```

## 📊 콘텐츠 템플릿

### 앵글 5가지

**1. 수수료_이슈**
- 예: "배민 수수료 인상", "쿠팡이츠 수수료 9.8%"
- 구성: 커버 → 문제 → 계산 → 비교 → 현실 → 대응 → CTA

**2. 할인_강요**
- 예: "배민페스타", "쿠팡이츠 슈퍼위크"
- 구성: 커버 → 유혹 → 실상 → 계산 → 딜레마 → 체념 → CTA

**3. 배달비_문제**
- 예: "배달비 인상", "무료배달 폐지"
- 구성: 커버 → 변경사항 → 점주 부담 → 계산 → 현실 → CTA

**4. 점주_피해**
- 예: "소상공인 폐업", "배달앱 종속"
- 구성: 커버 → 문제 → 통계 → 사례 → 원인 → 대안 → CTA

**5. 갑질_이슈**
- 예: "리뷰 테러", "일방적 계약 변경"
- 구성: 커버 → 사례 → 분노 → 실태 → 대응 → CTA
