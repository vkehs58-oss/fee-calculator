### 3단계: CapCut에서 합성

**레이어 구조:**
```
[상단] 토스 로고 (작게, 흰색)
[중앙] 메인 카피 (크고 굵게)
[하단] 서브 카피 + CTA
```

**토스 로고 배치:**
- 위치: 상단 중앙 또는 좌측 상단
- 크기: 가로 100-120px (전체 이미지의 10%)
- 여백: 상단에서 40-60px
